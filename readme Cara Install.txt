1. install xampp
	link:https://www.apachefriends.org/xampp-files/7.4.21/xampp-windows-x64-7.4.21-0-VC15-installer.exe
	
note:install seperti biasa tidak ada pengaturan khusus

2. download Source Code aplikasi SPP BASE
	cara: 
	klick "code" berwarna hijau -> download Zip
	ekstrak file "base-spp"
	simpan di folder c:/xampp/htdocs
	
3. jika ingin menjalankan aplikasi spp-base
	cara: jalankan xampp-control ---ada di c:/xampp
	
4. start Apache dan MySql   

5. buka google/browser lainnya

6. import database
	cara:
	- ketik link phpmyadmin: localhost/phpmyadmin/
	- buat database baru dengan nama "db_spp"
	- import , choose file db_spp.sql di directory c:/xampp/htdocs/spp-base ,lalu go.
	
7. link alikasi spp-base:localhost/spp-base/
